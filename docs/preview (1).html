ClientPulse AI

Account, contract and project health agent with retrieval over contracts, SOWs, SLA reports and manager status updates
Tech stack note: MVP uses Streamlit, Python, OpenAI Agents SDK, Responses API, File Search, vector store, function calling, and local SQLite/files. Enterprise version can add React + ChatKit, ChatGPT Workspace Agents, MCP connectors, SharePoint, Jira, ServiceNow, CRM, CLM, finance systems, Postgres or Databricks Delta. Keep OpenAI Agent Builder out of the core architecture because OpenAI is deprecating it, while ChatKit remains available.
Primary data: uploaded docs Optional integrations: only if available Core output: risk + confidence + action Human review required
